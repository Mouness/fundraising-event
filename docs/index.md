# Fundraising Event Project Documentation

Welcome to the documentation for the Fundraising Event project.

## Overview

This project is a real-time fundraising platform featuring:
- **NestJS Backend**: Handles donations, authentication, and WebSocket events.
- **React Frontend**: Provides the donation interface, admin dashboard, and live event screen.
- **Real-time Updates**: Live gauge and counters for event screens.
