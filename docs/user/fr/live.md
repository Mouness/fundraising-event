# Mode Live (Écran Géant)

L'**Écran Live** est une fonctionnalité puissante conçue pour être projetée sur de grands écrans ou des projecteurs lors de votre événement physique. Il agit comme un tableau de bord en temps réel qui visualise la progression de la collecte de fonds.

![Écran Live](../assets/live_screen.png)

## Fonctionnalités

- **Jauge de Progression** : Se remplit dynamiquement à mesure que vous approchez de l'objectif.
- **Flux de Dons** : Affiche les nouveaux dons instantanément lorsqu'ils arrivent (Nom & Montant).
- **Code QR** : Un code scannable qui mène directement à la page de don mobile, facilitant la participation du public.
- **Total Collecté** : Le montant total à jour.

## Comment Utiliser

1.  Ouvrez votre événement.
2.  Cliquez sur le lien **Page Live** dans le tableau de bord ou naviguez vers `/mon-evenement/live`.
3.  Passez votre navigateur en **Plein Écran** (F11).
4.  Connectez votre ordinateur au projecteur de la salle.

> [!TIP]
> L'écran Live se met à jour automatiquement. Vous n'avez pas besoin de rafraîchir la page. Il utilise des WebSockets pour une réactivité instantanée.
