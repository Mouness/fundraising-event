# Personnel & Bénévoles

Gérez votre équipe et donnez aux bénévoles les outils pour collecter des dons sur le terrain.

## Gestion du Personnel

1.  Accédez à **Paramètres > Équipe** (ou **Personnel** dans le menu admin global).
2.  Ajoutez un nouveau membre ou éditez un profil existant.
3.  Assignez un **Code PIN** (4 chiffres). Ce code est essentiel pour l'App Collecteur.

![Gestion du Personnel](../assets/staff_management.png)

## App Collecteur

L'application Collecteur est une interface simplifiée conçue pour les bénévoles sur le lieu de l'événement afin de collecter les dons rapidement via tablette ou mobile.

### 1. Accès & Connexion

Naviguez vers `[url-evenement]/staff/login` (ex: `/ocean-cleanup-2025/staff/login`).

![Connexion Collecteur](../assets/collector_login.png)

1.  Entrez votre **Code PIN à 4 chiffres** (assigné dans le Panneau Admin).
2.  Cliquez sur **Connexion**.

> [!NOTE]
> Exemple de PIN : `1010` (ou vérifiez votre profil staff).

### 2. Tableau de Bord

Une fois connecté, les bénévoles voient un tableau de bord épuré pour traiter les dons sans accéder aux fonctionnalités administratives sensibles.

![Tableau de Bord Collecteur](../assets/collector_dashboard.png)

### Accéder à l'App

1.  Ouvrez l'URL de l'événement sur l'appareil : `http://localhost:5173/ocean-cleanup-2025`
2.  Ajoutez `/staff` à l'URL.
3.  Connectez-vous avec le PIN.
